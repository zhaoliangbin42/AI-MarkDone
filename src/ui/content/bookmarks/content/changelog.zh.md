# 更新日志

# 4.1.0
2026-04-16

现在在阅读器中选中任意内容后，直接复制出来的是源码而不是渲染后的纯文本。这一版还正式带来了“灵动注释”：你可以一边阅读，一边把不满意的段落、句子、公式或代码逐条标出来，写下自己的修改意见，然后一键整理成适合发给大模型的输入。相比一次只处理一句的官方逐句批注方式，这套流程更适合文章级改写、批量润色和针对性回复，既更省对话次数，也更容易得到更准确的结果。

- 阅读器现在支持对选中内容直接添加注释。选中正文后，会在附近出现悬浮的复制和注释按钮。
- 注释会和对应选区建立关联，并在阅读器中以高亮方式持续标记出来。
- 已有注释支持再次打开查看和编辑，方便反复修改。
- 新增注释导出模板，可在“设置 > 阅读器 > 注释复制模板”中自定义每条注释的导出格式。
- 新增 Prompt Library，可在“设置 > 阅读器 > 用户提示词”中维护多条可复用的自定义提示词。
- 注释结果支持一键复制，入口位于阅读器顶部；当页面中已有注释内容时即可使用。
- 注释结果也可以直接插入阅读器左下角的发送消息框上方，减少手动复制粘贴。
- 阅读器现在支持对代码、公式、表格、图片等闭合内容按源码进行选择和复制。复制结果会尽量保持标准 Markdown 形式，而不再只是渲染后的纯文本。
- 书签面板新增 Changelog、About 和 FAQ 信息页，帮助你更快了解版本变化和项目背景。
- 优化了 unsupported 页面下的插件 popup，去掉了冗余内容，整体样式也更清晰。

# 4.0.0
2026-04-05

- 完成了一轮底层架构重构，整体行为更稳定，后续扩展也更清晰。
- 统一了主要面板、弹窗、按钮和提示的视觉风格，界面更协调，也更接近同一套产品语言。
- 页头书签入口和工具栏交互进一步统一，在不同平台上的位置和行为更稳定。
- Markdown 渲染链路做了统一，代码、公式、表格等内容在不同界面中的表现更加一致。
- 修复了 ChatGPT 消息获取被截断的问题，长内容的提取更完整。
- 修复了 Claude 和 Gemini 上工具栏注入不稳定的问题。
- 修复了发送弹窗与原生输入框之间的同步问题，输入体验更可靠。
- 修复了若干阅读器、书签和弹窗相关问题，整体使用体验更顺手。

# 3.0.0
2026-02-19

- 支持 ChatGPT、Gemini、Claude 和 DeepSeek 等主流 AI 对话平台。
- 提供专门的阅读器模式，把长对话整理成更适合连续阅读的页面。
- 支持将对话内容复制为结构更完整的 Markdown，方便整理、保存和二次使用。
- 提供字数统计能力，帮助快速判断回复长度和信息密度。
- 内置书签系统，可以保存、分类和回看重要对话。
- 提供设置面板，用来控制阅读器、书签和工具栏等主要功能。
- 支持导出 Markdown 和 PDF，方便归档与分享。
- 为 ChatGPT 提供消息折叠能力，支持折叠旧消息以及一键全部折叠或展开。
