## âœ¨ Features at a Glance

### ğŸ“ Click-to-Copy Math Formulas

- Supports click-to-copy for single formulas **even during real-time streaming output**.
- Automatically extracts LaTeX formulas from AI responses.
- Supports both **inline formulas** (`$...$`) and **block formulas** (`$$...$$`).
- The copied content is directly compatible with popular Markdown editors like Typora and Obsidian.

### ğŸ§¾ One-Click Formatted Copy for ChatGPT & Gemini Dialogues (Markdown)

- One-click copy of the **Markdown source** for the current complete dialogue.
- Automatically wraps formulas using `$.$` and `$$.$$`.
- Attempts to re-wrap formulas enclosed by `\[...\]`, `\(...\)` with `$...$` and `$$...$$`.
- Automatically removes citation markers, hyperlinks, and other noisy content generated by ChatGPT or Gemini, resulting in cleaner Markdown.

### ğŸªŸ Markdown Source Display Panel

- **Directly displays** the Markdown source of the current dialogue in a panel format.
- Copy exactly what you want by free selection or segment copying.
- Ideal for scenarios where you only need to copy "a small segment."

### ğŸ”„ Live Preview Real-time Re-rendering

- After parsing the Markdown, it is **re-rendered** within the panel.
- Provides a "reading experience reset" for "disaster areas" like **Deep Research** long texts, or complex mixtures of formulas, tables, and text.
- Crucially, even if some formulas fail to render or appear heavily highlighted in the original page, the preview area renders them correctly.
- Supports **fullscreen display** for focused reading without the page distraction.

### ğŸ“Š Word & Character Count (Precise CJK Handling)

- Special handling for Chinese, Japanese, and Korean (CJK): `1 CJK character = 1 word + 2 chars`.
- Real-time statistics for the current dialogue's word and character count.
- Automatically excludes content like code blocks and math formulas for a count closer to the actual "body length."

2. Unzip the file.
3. Open Chrome and navigate to: `chrome://extensions/`
4. Enable **Developer mode** in the top right corner.
5. Click "Load unpacked," and select the unzipped `dist/` folder.

## ğŸ—ï¸ Project Architecture

This extension is built on a modern web technology stack:
