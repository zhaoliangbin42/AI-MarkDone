# Domain 08 Checklist：性能与稳定性

## A. 启动与注入

- [ ] content script 注入后 1s 内工具栏可交互
- [ ] 首次打开书签面板无明显卡顿（主观 < 300ms）
- [ ] 控制台无持续报错/重试风暴

## B. 阅读器渲染

- [ ] 100+ 消息会话切换到阅读模式可在可接受时间内完成
- [ ] 代码块/公式密集内容渲染期间无明显冻结
- [ ] 翻页与定位按钮交互无明显掉帧

## C. 书签高体量操作

- [ ] 导入 200/3000 条书签后列表操作响应稳定
- [ ] 批量移动/删除不出现持续闪烁或错乱
- [ ] 展开/折叠多层文件夹时无明显延迟累积

## D. 导入导出路径

- [x] Markdown 导出（大对话）在可接受时间内完成
- [x] PDF 导出不会残留导出容器节点
- [x] 导入失败路径不会触发异常重试或卡死

## E. 回归记录

- [ ] 每轮记录：数据规模、机器环境、主观耗时与异常点
- [ ] 若性能明显退化，记录可复现最小样本与提交范围
